# Analisis-Pengaruh-Harapan-Lama-Sekolah-terhadap-Tingkat-Kemiskinan-di-Provinsi-Jawa-Timur
Repositori ini berisi proses data wrangling terhadap dataset kemiskinan dan Harapan Lama Sekolah (HLS) di Jawa Timur tahun 2020-2024. Proyek ini bertujuan untuk menghasilkan dataset yang bersih, terstruktur, dan siap digunakan untuk analisis lanjutan

## Dataset
Dataset yang digunakan mencakup data:
- Persentase penduduk miskin
- Harapan Lama Sekolah (HLS)
- Rentang waktu: **2020â€“2024**
- Level data: **per wilayah dan tahun**

**Dataset bersumber dari Badan Pusat Statistik (BPS) dan telah diolah oleh penulis.**  
Data mentah diperoleh dari publikasi resmi BPS dan selanjutnya diproses melalui tahapan data wrangling untuk memastikan konsistensi format, kelengkapan data, serta kesiapan data untuk analisis.

## Tahapan Data Wrangling
Tahapan data wrangling yang dilakukan dalam proyek ini meliputi:
1. Penggabungan data dari beberapa sumber dan periode waktu
2. Pembersihan data dari nilai kosong dan inkonsistensi
3. Standarisasi penamaan wilayah
4. Penyesuaian format data numerik
5. Validasi struktur dan kelengkapan data
6. Penyusunan dataset akhir dalam format tabel per wilayah dan tahun

Hasil akhir berupa dataset yang bersifat **model-ready** dan dapat digunakan kembali tanpa memerlukan pengolahan ulang data mentah.

## Catatan
Proyek ini merupakan bagian dari tugas akademik dan ditujukan untuk memenuhi tugas akhir mata kuliah data wrangling.
